# Random User Profile App

This project is a modern Random User Generator built with Vue .  
It fetches random user data from an external API and displays the information inside the user interface.

---

# What The App Does

The application generates and displays a random user profile.

It shows:

- User profile picture
- Full name
- Email address
- Phone number
- Country
- City
- Username

When the **Generate User** button is clicked, a new random user is fetched and displayed instantly.


---

# Features

- Random user generation
- Modern responsive UI
- API integration
- Dynamic data rendering
- Interactive hover animations
- Mobile-friendly design
